# Personal Portfolio

This project is a personal portfolio website showcasing my skills, projects, and contact information. The site is designed to provide potential employers, collaborators, and others with an interactive experience that highlights my journey and accomplishments as a developer.

## Table of Contents

- [Features](#features)
- [Technologies Used](#technologies-used)
- [Project Structure](#project-structure)
- [Getting Started](#getting-started)
- [Usage](#usage)
- [Contact](#contact)

## Features

- **Responsive Design:** Optimized for both desktop and mobile devices.
- **Navigation Bar:** Smooth navigation across sections, including Home, About, Skills, Projects, and Contact.
- **About Section:** Provides an overview of my background, education, and key skills.
- **Skills Section:** Displays skills with animated icons.
- **Projects Section:** Lists completed projects with links to GitHub repositories and live demos.
- **Contact Section:** Provides multiple contact methods, including links to social media and email.
- **Animations:** Smooth fade-in and zoom effects for different sections.

## Technologies Used

- **HTML5**: Structure of the website
- **CSS3**: Styling, animations, and responsive design
- **JavaScript**: Interactivity and animations
- **Font Awesome**: Icons for social media links and skills
- **Google Fonts**: Custom fonts to enhance design

## Project Structure

```
- index1.html        # Main HTML file containing the structure of the website
- styles.css         # CSS file for styling, animations, and responsiveness
- script.js          # JavaScript file for interactivity and animations
```

## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Kalpeshgunjal07/portfolio-website.git
   ```

2. **Open the Project**
   - Navigate to the project directory and open `index1.html` in a browser.

## Usage

- **Navigation:** Click on navbar links to jump to different sections.
- **Project Demos and Repos:** Visit GitHub repositories and demos by clicking the "GitHub" and "Demo" buttons in the Projects section.
- **Contact Information:** Connect via the social media icons and view/download my resume.
